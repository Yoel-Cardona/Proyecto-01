Descripción del sistema

Este proyecto consiste en el desarrollo de un sistema en consola que simula la toma de decisiones dentro de una plataforma de streaming.

El programa permite evaluar contenidos antes de ser publicados, tomando en cuenta diferentes factores como el tipo de contenido, duración, clasificación, horario y nivel de producción.

A partir de estos datos, el sistema determina si el contenido puede ser publicado, enviado a revisión o rechazado, aplicando una serie de reglas previamente definidas.

Además, el sistema mantiene estadísticas de todas las evaluaciones realizadas durante la ejecución.

Instrucciones para ejecutarlo

Para ejecutar el programa se debe seguir los siguientes pasos:

Abrir el proyecto en Visual Studio o cualquier entorno compatible con C#.
Asegurarse de tener instalado .NET.
Ejecutar el programa (F5 o botón de iniciar).
Utilizar el menú interactivo que aparece en pantalla.
Seleccionar una opción ingresando el número correspondiente.

El programa se ejecuta en consola, por lo que toda la interacción se realiza mediante texto.

Breve explicación del proyecto

Este proyecto fue desarrollado como parte del curso de Pensamiento Computacional, con el objetivo de aplicar conceptos fundamentales de programación.

Durante su desarrollo se utilizaron estructuras como:

switch para el menú principal
if y if anidados para la toma de decisiones
ciclos como do-while, while y for
variables para el control de datos y estadísticas

El sistema evalúa cada contenido aplicando reglas técnicas, luego calcula su nivel de impacto y finalmente toma una decisión basada en estos resultados.

Este proyecto permitió comprender cómo se pueden resolver problemas reales mediante programación, así como la importancia de la lógica en la toma de decisiones dentro de un sistema.
